About this website

This design is the 5th iteraiton of PatrickDoran.com. This mobile-first site was designed in the browser, built entirely on my Chromebook, using a great text editor called Zed.

  • I started writing its HTML5 using best practices from HTML5 Boilerplate and Google Web Starter Kit.
  • The images are served from a different domain to decrease page load time.
  • The colors used are inspired by FlatColors.
  • The SVG icons come from Customizr.net.
  • Custom fonts Porter Sans and Amble are loaded, but just for larger screen sizes.

I used Angular.JS to load portfolio data from JSON file, and its routing feature to create per-project views with unique URLs. The reason was to learn how to use a client side MVC framework, and the project data seemed like a good source. While it was a lot of fun to learn, I don't think Angular's the right choice for a portfolio site. The download tax for moblie is pretty high and the javascriptless suffer.
